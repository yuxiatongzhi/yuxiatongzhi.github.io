      • setVideoScaleRatio

        int setVideoScaleRatio(float horizontal,
                       float vertical)
        设置视频画面在背景画布上的缩放比

        horizontal - 水平缩放比
        vertical - 垂直缩放比
        返回:
        0:success -1:fail

      • setVideoScaleRatio

        int setVideoScaleRatio(float horizontal,
                       float vertical,
                       boolean render)
        设置视频宽高和底图宽高比

        note: 此接口的调用的前提是 有背景 IRender.setBackgroundBitmap(String, boolean, boolean) 或者 IRender.enableBackgroundBlur(boolean, boolean) 开启模糊，如果无背景则设置无效

        参数:
        horizontal - 0.0f - 1.0f
        vertical - 0.0f - 1.0f
        render - 是否立即渲染
